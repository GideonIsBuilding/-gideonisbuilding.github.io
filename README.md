# Gideon — Portfolio

Dark terminal-style portfolio site. Built with vanilla HTML, CSS, and JS.

---

## Project Structure

```
gideon-portfolio/
├── index.html      ← main page (all sections)
├── style.css       ← all styling (CSS variables, layout, responsive)
├── main.js         ← git grid, scroll spy, mobile sidebar
└── README.md       ← you are here
```

## Fonts

Loaded from Google Fonts (requires internet connection):
- **JetBrains Mono** — body, code, UI elements
- **Syne** — display name heading

For offline use, download both from [fonts.google.com](https://fonts.google.com) and update the `@import` in `style.css` to point to local files.
